C++ type: QStyle::StyleHint

C++ documentation:

This enum describes the available style hints. A style hint is a general look and/or feel hint.

See also styleHint().

Variants

Base value for custom style hints. Custom values must be greater than this value. (C++ enum variant: SH_CustomBase = -268435456)
