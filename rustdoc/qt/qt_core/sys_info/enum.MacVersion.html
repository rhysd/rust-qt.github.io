C++ type: QSysInfo::MacVersion

C++ documentation:

This enum provides symbolic names for the various versions of the Darwin operating system, covering both OS X and iOS. The QSysInfo::MacintoshVersion variable gives the version of the system on which the application is run.

See also WinVersion.
