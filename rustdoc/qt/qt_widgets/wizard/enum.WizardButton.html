C++ type: QWizard::WizardButton

C++ documentation:

This enum specifies the buttons in a wizard.

The following value is only useful when calling setButtonLayout():

See also setButton(), setButtonText(), setButtonLayout(), and customButtonClicked().

Variants

NoButton

C++ enum variant: NoButton = -1

BackButton

The Back button (Go Back on Mac OS X) (C++ enum variant: BackButton = 0)

NextButton

The Next button (Continue on Mac OS X) (C++ enum variant: NextButton = 1)

CommitButton

The Commit button (C++ enum variant: CommitButton = 2)

FinishButton

The Finish button (Done on Mac OS X) (C++ enum variant: FinishButton = 3)

CancelButton

The Cancel button (see also NoCancelButton) (C++ enum variant: CancelButton = 4)

HelpButton

The Help button (see also HaveHelpButton) (C++ enum variant: HelpButton = 5)

CustomButton1

This variant corresponds to multiple C++ enum variants with the same value:

  • CustomButton1 = 6: The first user-defined button (see also HaveCustomButton1)
  • NStandardButtons = 6
CustomButton2

The second user-defined button (see also HaveCustomButton2) (C++ enum variant: CustomButton2 = 7)

CustomButton3

The third user-defined button (see also HaveCustomButton3) (C++ enum variant: CustomButton3 = 8)

Stretch

This variant corresponds to multiple C++ enum variants with the same value:

  • Stretch = 9: A horizontal stretch in the button layout
  • NButtons = 9
